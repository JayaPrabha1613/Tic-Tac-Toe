# Tic-Tac-Toe
A beginner friendly java game developement

Description:
A classic 2-player Tic-Tac-Toe game built using Java Swing. The game provides a simple and interactive GUI for two players to compete, detects winning conditions, and allows easy restarting.

Features:

Grid Layout: 3x3 button grid for gameplay
Turn-Based Play: Alternates between X and O
Winning Detection: Automatically checks for wins in rows, columns, and diagonals
Draw Detection: Alerts when the game ends in a draw
Restart Option: Reset the board to play again
User-Friendly Interface: Simple and responsive GUI

Technologies Used:

Java (JDK 8+)
Swing for GUI

How to Run:

Compile the Java file: javac TicTacToe.java

Run the game: java TicTacToe
